The system consist of 3 modules: 
OFIRCS- an online FIR lodging system for public.
CISPS- a criminal information system for police departments to manage FIR and Criminal records.
SP- an Administration system for Police Department.

The system was developed as Final Year Project for my Bachelor in Computer Science in Pakistan.
Hence, locations for police departments in Karachi, Pakistan has been used as a sample database.
Criminal records are all dummy data and does not violate any security rules.
